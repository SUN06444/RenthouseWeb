# RenthouseWeb 學生租屋網

## 1. 系統說明:

本系統提供安全保障、地圖探索、房屋比較等功能，目的在於，提供給不論是大學新鮮人，還是學長姐，只要是學生，都能透過線上看房，去找到適合自己租屋的地方。

## 2. 開發環境:

(a)MAMP網頁伺服器架站(Apache、Mysql、php)

(b)Sublime Text(文字編輯器)

(c)網域.tech及PLESK免費主機

## 4. 功能項目

- 註冊功能(房客、房東)、登入功能(房客、房東、管理人員)

- 房屋搜尋功能

- 房客：房屋收藏、房屋比較功能、基本資料管理、收藏紀錄查詢

- 房東：刊登房屋功能

- 管理人員：審核學生身分以及審核房屋刊登資訊

- 警局、醫院、診所分布圖
